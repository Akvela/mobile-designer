# Мобильный дизайнер
Учебный проект от Яндекс Практикума в рамках проектного месяца. В нем используются технологии:

* Препроцессор SASS
* Java Script
* Grid Layout
* Flex
* Свайпер
* Плавность переходов
* Разметка и верстка форм

Страница про мобильное приложение для плохих художников.
